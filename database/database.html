WinForms ja Entity Framework Core Rakendus

Rakenduse Kirjeldus

See rakendus võimaldab kasutajatel hallata lihtsat andmebaasi, kasutades WinForms kasutajaliidest ja Entity Framework Core-i andmebaasitoiminguteks. Rakendus on loodud andmete loomiseks, lugemiseks, muutmiseks ja kustutamiseks (CRUD).

Rakenduse Peamised Funktsioonid

  • Andmebaasiühenduse loomine ja haldamine.
  • CRUD-toimingud (andmete lisamine, vaatamine, muutmine, kustutamine).
  • Kasutajasõbralik graafiline kasutajaliides (GUI), mis toetab intuitiivset kasutuskogemust.
  • Andmebaasimudelite automaatne loomine ja kasutamine EF Core abil.
  • Andmete kuvamine tabelis ja vormi vaadetel (WinForms DataGridView).

Süsteeminõuded

  • Windows 10 või uuem operatsioonisüsteem.
  • .NET 6.0 SDK või uuem.
  • Visual Studio koos "Desktop Development with .NET" tööriistadega.
  • Microsoft SQL Server andmebaas või muu toetatud EF Core andmebaas.

Tehnoloogiad

  • WinForms: Kasutajaliidese loomiseks.
  • Entity Framework Core: Andmebaasitoiminguteks ja mudelite haldamiseks.
  • SQL Server: Andmebaasi hoidmiseks ja haldamiseks.

Kasutusvaldkonnad

  • Lihtsad ärirakendused, näiteks inventuuri haldus.
  • Andmebaasi halduse tööriistad.
  • Isiklikud projektid ja õppematerjalid andmebaasirakenduste loomiseks.

Käesolev rakenduse nõuete leht on koostatud näitena ja põhineb üldistel EF Core ja WinForms võimalustel.
